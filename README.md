# Dotfiles
Various .rc files for my command line experience


## Installation
    git clone https://github.com/ajh17/dotfiles.git
    chmod +x ~/dotfiles/bootstrap.zsh && ~/dotfiles/bootstrap.zsh

- Terminal : zsh + tmux
- Editor   : Vim
- IRC      : Weechat
