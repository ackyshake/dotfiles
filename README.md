# Dotfiles
Various .rc files for my command line experience

## Installation
    git clone --recursive https://github.com/ajh17/dotfiles.git
    chmod +x ~/dotfiles/bootstrap && ~/dotfiles/bootstrap

## My Environment:
- OS       : OS X
- Terminal : zsh + tmux
- Editor   : Vim
- IRC      : Weechat
